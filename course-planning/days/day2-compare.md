# Day 2 Compare
# Day 2 比較表 - AI 工具的選擇與搭配

## Core Goal

Help learners stop asking "Which AI is best?" and start choosing tool combinations based on task structure.

讓學員不再問「哪個 AI 最強」，而是學會依任務選工具組合。

## Version Comparison

| Version | Core Design | Strength | Risk | Keep / Modify / Drop |
|---|---|---|---|---|
| Tool map version | Introduces common tools by category | Easy to understand | Can become a tool list | TBD |
| AI Tool Picker version | Three-question decision framework | Practical and memorable | Needs examples | TBD |
| Case-library version | 3-5 workflow templates | Strong transfer value | Needs careful preparation | TBD |

## Candidate Final Elements

- Three-question AI Tool Picker:
  - Exploration or production?
  - Need latest information?
  - Need long multi-step workflow?
- Tool combination case library.
- Reverse challenge: how would you do this without AI?
- Trap warnings:
  - Confidential data without anonymization
  - Unverifiable output
  - High legal, medical, or financial risk
  - Prompt setup cost higher than manual work
  - Sensitive trust or relationship handling
- Learner output: task decomposition canvas with human review points.
