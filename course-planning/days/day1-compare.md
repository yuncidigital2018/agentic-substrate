# Day 1 Compare
# Day 1 比較表 - 重新認識生成式 AI

## Core Goal

Help learners move from "AI is a chat tool" to "AI is a new work-system layer."

讓學員從「AI 是聊天工具」升級到「AI 是新的工作系統入口」。

## Version Comparison

| Version | Core Design | Strength | Risk | Keep / Modify / Drop |
|---|---|---|---|---|
| Student feedback version | Game lobby + live experience + personal website | High participation and energy | May become too tool-demo oriented | TBD |
| Agentic Substrate version | Three-layer model + shock demo | Strong cognitive shift | May feel abstract if not embodied | TBD |
| Integrated version | Safety red lines + 30-person event demo + instant experience | Universal, practical, safer | Needs careful demo prep | TBD |

## Candidate Final Elements

- Three safety red lines in the first 3 minutes.
- Universal shock demo: 30-person event planning.
- Simplified model: `LLM -> Tools / Agents -> Workflow`.
- Process-cost observation sheet.
- Short instant experience task.
- Close by previewing Day 2: choosing tool combinations.
