# Course Planning Workspace
# 課程規劃工作區

This folder is for comparing multiple versions of the three-session generative AI course and gradually converging on a final version.

這個資料夾用來比較三堂生成式 AI 課程的不同版本，並逐步整合成最終版。

## Folder Structure

```text
course-planning/
|-- README.md
|-- dashboard.md
|-- decision-log.md
|-- versions/
|   |-- _paste-versions-here.md
|-- days/
|   |-- day1-compare.md
|   |-- day2-compare.md
|   `-- day3-compare.md
|-- final/
|   |-- day1-final.md
|   |-- day2-final.md
|   |-- day3-final.md
|   `-- full-course-final.md
`-- templates/
    |-- version-template.md
    |-- day-compare-template.md
    `-- final-day-template.md
```

## Workflow

1. Paste each source version into `versions/`.
2. Compare each day in `days/day*-compare.md`.
3. Record major decisions in `decision-log.md`.
4. Move stable decisions into `final/`.

## Course Anchor

> 你不是在用 AI，你是在設計一個會幫你工作的系統。

Three-day narrative:

- Day 1: AI is a system.
- Day 2: You are designing the system.
- Day 3: You become the system architect.
